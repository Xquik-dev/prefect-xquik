# Security Policy

Do not include Xquik API keys or other credentials in issues, pull requests,
examples, logs, or tests.

Report security issues through GitHub private vulnerability reporting for this
repository. If private reporting is unavailable, email
[support@xquik.com](mailto:support@xquik.com).

Do not open public issues containing exploit details, secrets, or private data.

We aim to acknowledge reports within 3 business days. We will coordinate a
disclosure timeline after confirming the issue.

Security fixes target the latest published release.
